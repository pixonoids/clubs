Description:

Objective:

In this project, we aim to replicate the NIT Hamirpur campus in a virtual world, so that all those who are willing to see the campus will be able to explore as well as remember their memories with the campus.

About This Project:

Due to the pandemic, the college campus was off-reach for 1 year so the students couldn't visit their campus throughout the year. With the help of our app, in only 30 MB, you can see Indiaâ€™s one of the most beautiful campus.

Software used:

We have made it by using multiple softwares such as Blender, Unity and Visual Studio Code.

Execution:

Firstly we made 3D models of the NIT Hamirpur campus (Building, trees, Roads, playground, etc.). Then we exported it in the (.fbx) format which was then imported to unity. Then we added many assets (eg. 3rd person character, joystick in the mobile version, etc) and scripts (eg. scene changing, 3rd person character, 3rd person movement, joystick script, volume button, etc ). Then we added some textures. Then we created UI of the scenes.

Novelty:

We also created a natural atmosphere by linking the game time with the device time so that the day/night will occur according to the time in the VR simulation. In the end, all the scripts were processed in Visual Studio Code.

Business aspects:

We can launch this app as a virtual trip to NIT Hamirpur and also have a plethora of other uses of the app.
